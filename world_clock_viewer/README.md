# World Clock Viewer ⏰

A simple Firefox extension to view the current time in multiple cities around the world.

## Features
- Add cities with timezones
- Customize display order
- Lightweight, no tracking or permissions
- Built for remote workers, travelers, and global teams

## Install
1. Download the repository as `.zip`
2. Go to `about:debugging` in Firefox
3. Unzip
4. Click “Load Temporary Add-on”, and select the `manifest.json` file
